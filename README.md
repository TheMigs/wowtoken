# WoW Token
A solo project, loading JSON data into an Android application to track real-time pricing of a World of Warcraft Token.

Application implements:
- Minimalistic Material UI Design
- Loaders
- Fragments
- ViewPagers
- TabLayouts
- Adapters


This application is not affliated with any official Activision Blizzard entities. 

It is an unofficial app implementing an unofficial service provided by <a href="https://wowtoken.info">wowtoken.info</a>
